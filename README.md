# 💰 Expense Tracker (Python + File Handling)

## 📌 Project Description
This is a simple Expense Tracker project built using Python and File Handling. It helps users manage daily expenses by adding, categorizing, storing, and analyzing their spending in a text file.

## 🚀 Features
- Add daily expenses  
- Category system (Food, Travel, Recharge, Other)  
- Store data permanently using file handling  
- View saved expenses from file  
- Calculate total expense, saving, highest expense, and number of entries  

## 🛠️ Technologies Used
- Python  
- File Handling  
- OOP (Class & Methods)  

## 📂 How It Works
User enters pocket money, adds multiple categorized expenses, data is saved in a file, and a final summary is displayed including total expense, saving, and highest expense. Then all stored data is read from the file and shown.

## 🎯 Learning Outcomes
- Python basics and logic building  
- File handling (read/write/append)  
- OOP concepts  
- Real-world problem solving
